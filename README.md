# Roman Bharati - Personal Portfolio

A modern, finance-themed personal portfolio website for Roman Bharati.

## Pages
- **Home**: Hero section with name, university, and contact info, featuring typing animation.
- **Experience**: Professional work experience.
- **Projects**: Personal and academic projects.
- **Essays**: Placeholder for essays (coming soon).

## Technologies Used
- HTML5
- CSS3 (with animations and modern design)
- JavaScript (for typing effect and smooth scrolling)

## Theme
Subtle, clean, professional design inspired by finance and fintech, with dark blue gradients, gold accents, and subtle crypto-inspired patterns.

## How to View
Open `index.html` in a web browser or run a local server (e.g., `python3 -m http.server 8000`).

## Features
- Responsive design
- Smooth animations (fade-in, slide-up, typing effect)
- Fixed navigation header
- Hover effects on cards
